# Jogo do livro "Python Crash Course"

## Olá, tudo bem com você?

Esse jogo não é meu, é apenas um exercício de um livro que ensina Python.
O jogo é feito com Python e Pygame.
Para jogá-lo é preciso baixar os arquivos e executá-lo.
Esse código foi testado em um ambiente Linux, mais especificamente o Ubuntu 22.04.
Por isso, não garanto seu funcionamento em ambientes Windows, outros ambientes Linux ou Mac.
